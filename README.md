## 🧩 Funcionalidades

- 🧾 Geração automática de cartelas em formato PDF
- 🧠 Interface moderna com `CustomTkinter`
- 🎯 Suporte a modelos SVG e planilhas Excel
- 📤 Exportação em lote com barra de progresso
- ⛔ Cancelamento da geração com segurança
